nextjs无法处理服务端
- 数据接口
- 数据库链接
- session状态

为什么Koa
- Koa是一个非常轻量级的 nodejs 服务端框架
- 项目相对轻量，并不需要很多集成的功能 
- 非常易于扩展，编程模式非常符合js特性

Koa特点
- 轻量：本身不封装什么功能

Koa api
- app.use
- ctx对象
- request、response、req、res  的关系
